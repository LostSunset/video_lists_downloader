# 快速入門指南

本指南將協助您在 5 分鐘內開始使用影片批量下載工具。

## 系統需求

- Python 3.10 或更高版本
- PySide6
- yt-dlp
- Firefox 瀏覽器（用於 Cookie 提取，選用）

## 安裝

```bash
# 使用 uv（推薦）
uv sync

# 或使用 pip
pip install PySide6 yt-dlp
```

## 三步驟快速開始

### 1️⃣ 選擇下載路徑
- 點擊左側「下載路徑」的「瀏覽」按鈕
- 選擇要儲存影片的資料夾
- 快捷鍵：`Ctrl+O`

### 2️⃣ 輸入影片網址
- **單一影片**：直接貼上影片網址
- **播放清單**：貼上含有 `list=` 的播放清單網址
- **批量下載**：選擇包含多個網址的 `.txt` 檔案

> 💡 **小技巧**：可直接拖放 URL 或檔案到視窗！

### 3️⃣ 開始下載
- 點擊「開始下載」按鈕
- 快捷鍵：`Ctrl+Enter`

## 鍵盤快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl+Enter` | 開始下載 |
| `Ctrl+Shift+V` | 快速貼上 URL |
| `Ctrl+O` | 選擇下載路徑 |
| `F1` | 顯示說明 |
| `Ctrl+Q` | 退出程式 |

## 下載會員影片

如需下載 YouTube 或 Bilibili 的會員限定影片：

1. 先在 Firefox 瀏覽器登入 YouTube 或 Bilibili
2. 關閉所有 Firefox 視窗
3. 點擊對應平台的「提取」按鈕
4. 點擊「測試」確認 Cookie 有效（顯示 🟢）
