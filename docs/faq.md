# 常見問題 (FAQ)

## Cookie 相關

### Q: Cookie 提取失敗？

**解決方案：**
1. 確認已在 Firefox 瀏覽器登入 YouTube/Bilibili
2. **完全關閉**所有 Firefox 視窗（包括背景程序）
3. 重新點擊「提取」按鈕

### Q: Cookie 狀態顯示 🔴 無效？

**可能原因：**
- Cookie 已過期（通常 30 天後失效）
- 瀏覽器登入狀態已變更
- Cookie 檔案損壞

**解決方案：**
1. 在 Firefox 重新登入平台
2. 關閉 Firefox 後重新提取 Cookie

### Q: 為什麼需要 Cookie？

Cookie 用於下載需要登入才能觀看的影片，例如：
- YouTube 會員專屬影片
- Bilibili 大會員影片
- 年齡限制影片

## 下載相關

### Q: 下載失敗顯示「格式不可用」？

**解決方案：**
- 嘗試將畫質設為 `best`（自動選擇最佳可用格式）
- 確認影片未被刪除或設為私人

### Q: 下載速度很慢？

**可能原因：**
- 網路連線問題
- 來源伺服器限制
- 同時下載太多影片

**建議：**
- 減少同時下載數量
- 檢查網路連線
- 嘗試在不同時間重新下載

### Q: 跳過已下載的影片？

程式會自動偵測已下載的影片並跳過，判斷依據：
1. 下載歷史記錄
2. 檔案名稱中的影片 ID（如 `[dQw4w9WgXcQ]`）

## 字幕相關

### Q: 沒有下載到字幕？

**可能原因：**
- 影片沒有可用字幕
- 指定的語言不存在

**建議：**
- 勾選「下載自動生成字幕」
- 擴展語言設定（例如：`zh-TW,zh,zh-Hant,zh-Hans,en`）

### Q: 字幕語言代碼是什麼？

常用語言代碼：
| 代碼 | 語言 |
|------|------|
| `zh-TW` | 繁體中文（台灣） |
| `zh-Hant` | 繁體中文 |
| `zh-Hans` | 簡體中文 |
| `zh` | 中文 |
| `en` | 英文 |
| `ja` | 日文 |
| `ko` | 韓文 |

## 其他問題

### Q: 程式無法啟動？

確認已安裝所有相依套件：
```bash
pip install PySide6 yt-dlp
```

### Q: 如何更新 yt-dlp？

程式啟動時會自動更新，也可手動更新：
```bash
pip install --upgrade yt-dlp
```
