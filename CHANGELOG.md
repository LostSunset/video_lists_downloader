# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-01-26

### Added
- 下載速度限制功能
- 自動重試機制
- 系統托盤支援
- 下載完成通知
- 深色/淺色主題切換
- 下載統計功能
- 設定匯出/匯入功能

### Changed
- 從 PyQt6 遷移到 PySide6
- 預編譯正規表達式提升效能

### Removed
- 移除未使用的 import 與空類別

## [0.1.9] - 2026-01-20

### Added
- Bilibili 平台支援
- 播放清單批量下載

## [0.1.0] - 2026-01-01

### Added
- 初始版本
- YouTube 影片下載
- 基本 GUI 介面
